## 二、快速开始

### 第一步：填写人员信息

在顶部工具栏右侧的输入框中填写：

- **采集员**：本次数据的采集人员姓名
- **筛选员**：执行本次筛选操作的人员姓名（同一筛选员在整个会话中只需填写一次）

> 筛选员姓名在切换 Episode 时不会被覆盖；采集员姓名会随 Episode 切换自动更新（若该 Episode 已有保存记录）。

### 第二步：选择数据文件夹

点击左上角「📂 选择文件夹」按钮，选择包含若干 `episode_XXXX` 子文件夹的根目录。

**期望的文件夹结构：**

```
数据根目录/
├── episode_0000/
│   ├── colors/
│   │   ├── 000000_color_0.jpg
│   │   ├── 000001_color_0.jpg
│   │   └── ...
│   ├── audios/
│   ├── depths/
│   └── data.json
├── episode_0001/
│   └── ...
└── ...
```

> 工具自动识别所有以 `episode_` 开头的子文件夹，并按名称排序加载。

## 三、图片浏览

| 操作 | 方式 |
|------|------|
| 下一张图片 | 键盘 `D` 或点击「下一张」按钮 |
| 上一张图片 | 键盘 `A` 或点击「上一张」按钮 |
| 切换 Episode | 顶部下拉框选择，或使用底部「上一个 / 下一个 Episode」按钮 |

- 图片计数器（如 `5 / 42`）显示当前帧编号和总帧数
- 图片会自动缩放以适应窗口大小，拖动窗口边缘可调整显示比例

## 四、打标签

### 4.1 任务标签（单选）

位于标签面板顶部，**每个 Episode 只能选择一个任务**，共三类：

| 任务类别 | 可选项 |
|----------|--------|
| 抓放任务 | 简单抓放任务 / 分类任务 |
| 擦拭任务 | 擦拭任务 |
| 锤击任务 | 锤击任务 |

点击单选按钮（○）即可选中，同一类别中的选项互斥。

### 4.2 物体标签（多选）

记录机器人操作的主体物体：

| 分类 | 子类 | 可选项 |
|------|------|--------|
| 刚性物体 | 水果(fruit) | 无 / 苹果 / 梨子 / 香蕉 / 杨桃 / 黄瓜 / 柿子 |
| | 方块(cube) | 无 / 魔方 / 积木 / 充电器 |
| | 工具(tool) | 无 / 锤子 / 钳子 |
| | 可抓取容器 | 无 / 马克杯 / 可乐瓶 / 易拉罐 |
| 柔性物体 | 布(fabric) | 无 / 抹布 / 毛巾 |
| | 衣物(clothing) | 无 / 衬衫 / T恤 / 裤子 |
| | 清洁工具 | 无 / 海绵 |

### 4.3 容器标签（多选）

记录任务中装载物体的容器：

| 分类 | 子类 | 可选项 |
|------|------|--------|
| 刚性容器 | 三维容器 | 无 / 菜篮子 / 收纳盒 / 三层置物架 |
| | 二维容器 | 无 / 盘子 / 砧板 / 杯垫 |
| | 承载液体的容器 | 无 / 马克杯 / 塑料瓶 |
| 柔性容器 | 布料容器 | 无 / 帆布袋 |
| | 塑料容器 | 无 / 塑料袋 |

### 4.4 背景标签（多选）

记录任务场景的环境信息：

| 分类 | 子类 | 可选项 |
|------|------|--------|
| 光照 | 强度 | 无 / 早全灯 / 晚全灯 / 早半灯 / 晚半灯 |
| | 颜色 | 无 / 暖色 / 冷色 |
| 桌面材质 | — | 木头材质 / 金属材质 / 塑料材质 |
| 背景物体 | 二维物体 | 无 / 砧板 / 桌垫 / 杯垫 |
| | 三维物体 | 无 / 三层置物架 / 菜篮子 |
| | 零散物体 | 无 / 水果 / 工具 / 方块 |

> 每个子类中的「无」选项表示该类别中没有任何相关物体。

## 五、保存与导航

### 保存标签

| 操作 | 方式 |
|------|------|
| 保存当前 Episode 的标签 | 键盘 `Space` 或点击「💾 保存标签」按钮 |

- 保存成功后自动跳转到**下一个 Episode**
- 若当前已是最后一个 Episode，状态栏提示「已是最后一个 Episode」
- 保存文件名固定为 `flag.json`，保存在当前 Episode 文件夹内

### 标签继承机制

- 切换到**已有 `flag.json`** 的 Episode 时：自动清空并加载该文件中的标签
- 切换到**尚未保存**的 Episode 时：保留上一个 Episode 的标签选项作为起点，方便连续筛选相似数据
- **仅在点击保存时才写入文件**，切换 Episode 不会自动保存

### 删除轨迹

| 操作 | 方式 |
|------|------|
| 删除当前 Episode | 键盘 `Q` 或点击「🗑 删除轨迹」按钮 |

> 删除操作会弹出二次确认对话框，确认后**永久删除整个 Episode 文件夹**（包含图片、音频、深度图等所有数据），**不可撤销**，请谨慎操作。

删除后自动跳转到下一个 Episode；若删除的是最后一个，则跳转到新的末尾。

## 六、快捷键汇总

| 按键 | 功能 |
|------|------|
| `A` | 上一张图片 |
| `D` | 下一张图片 |
| `Space` | 保存标签并跳转下一个 Episode |
| `Q` | 删除当前 Episode（需二次确认） |

## 七、输出文件格式

保存后在每个 Episode 文件夹下生成 `flag.json`，格式示例：

```json
{
  "episode": "episode_0000",
  "采集员": "张三",
  "筛选员": "李四",
  "任务": "简单抓放任务",
  "物体": {
    "刚性物体": {
      "水果(fruit)": ["苹果", "梨子"],
      "可抓取容器": ["马克杯"]
    }
  },
  "容器": {
    "刚性容器": {
      "承载刚体的容器": {
        "二维容器": ["盘子"]
      }
    }
  },
  "背景": {
    "光照": {
      "强度": ["早全灯"],
      "颜色": ["暖色"]
    },
    "桌面材质": ["木头材质"],
    "背景物体": {
      "零散物体": ["水果"]
    }
  }
}
```

> 未勾选的标签分类不会写入文件，只记录实际选中的项。

## 八、注意事项

1. **首次使用**：打开工具后请先填写筛选员姓名，再选择文件夹
2. **采集员信息**：不同 Episode 的采集员可能不同，每次切换后请核对或修改
3. **删除操作不可撤销**：删除前请确认已完成该 Episode 的所有数据记录
4. **已筛选进度**：带有 `flag.json` 的 Episode 表示已完成筛选，重新保存会覆盖原记录
